# ZenPanel (Solid)

Admin UI shell for Solid + Vite + `@solidjs/router` + Tailwind.

Based on the latest Vite Solid scaffold (`solid-js@^1.9`, `vite@^8`, `vite-plugin-solid`).

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

Open [http://localhost:5173/admin/login](http://localhost:5173/admin/login).

Preview credentials: `admin` / `admin`.

## Customize

Edit `src/config/admin.config.ts` for branding and sidebar navigation.
