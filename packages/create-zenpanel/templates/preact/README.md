# ZenPanel (Preact)

Admin UI shell for Preact + Vite + React Router (via `preact/compat`) + Tailwind.

Based on the latest Vite Preact scaffold (`preact@^10`, `vite@^8`, `@preact/preset-vite`).

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

Open [http://localhost:5173/admin/login](http://localhost:5173/admin/login).

Preview credentials: `admin` / `admin`.

## Customize

Edit `src/config/admin.config.ts` for branding and sidebar navigation.
